# Domain 7 - Security Operations (13%)

This is the "keep the lights on and respond when they go out" domain. It covers the day-to-day work of running security: detecting and handling incidents, investigating and preserving evidence, recovering systems after a disaster, and controlling changes so they don't cause the next incident. At 13% it is one of the heavier domains, and it ties tightly to BCP from Domain 1 and the monitoring tools from Domains 4 and 6. Use the checklist below to navigate the topic notes.

## Domain Summary

This domain focuses on:

- **Incident management** - detecting, responding to, and recovering from incidents
- **Forensics** - investigating incidents and preserving evidence
- **Disaster recovery** - restoring operations after a disaster
- **Change management** - controlling changes to prevent incidents
- **Operations** - day-to-day security practices
